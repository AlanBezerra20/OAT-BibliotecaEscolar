Projeto de uma biblioteca escolar desenvolvido como uma aplicação SPA, utilizando o padrão MVC.

## 🌐 Aplicação online

[![Acessar aplicação](https://img.shields.io/badge/Acessar%20aplicação-Online-2ea44f?style=for-the-badge)](https://oat-biblioteca-escolar.netlify.app)

## ✨ Funcionalidades

- Página inicial, sobre, produtos e contato
- Cadastro e login de usuários
- Catálogo de produtos e serviços
- Pesquisa de produtos
- Painel administrativo
- Cadastro, edição e exclusão de registros
- Integração com Supabase
- Layout responsivo

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Supabase
- Netlify

## 📌 Projeto acadêmico

Desenvolvido para a disciplina de **PROGRAMAÇÃO PARA WEB II**, aplicando os conceitos de SPA, MVC, autenticação e banco de dados.
